## 🔑 需要你提供才能做的

| 我能做的事 | 需要你提供什么 |
|-----------|-------------|
| 调任何 API（高德、和风天气、OpenAI 等） | **API Key** 写进 `.env` |
| 操作 GitHub（PR/Issue/Release） | **GitHub Token** 或 `gh auth login` |
| 部署到云服务器 | **服务器 IP/账号** 或云服务商 Key |
| 发邮件/短信 | **邮件/短信服务商 API Key** |
| 操作数据库 | **数据库连接串** |
| 调微信/飞书/钉钉 API | **企业应用的 AppID/Secret** |
| 发布 npm/PyPI 包 | **npm/pip 的 publish token** |

> 一句话：**所有需要认证的外部服务，你给 Key 我就能调，你不给我就调不了。**

## ❌ 我做不到的

### 账号相关
| 做不到 | 原因 |
|--------|------|
| 帮你注册任何账号 | 需要手机号/身份证/人脸识别 |
| 帮你实名认证 | 需要你的生物特征 |
| 帮你绑卡/支付 | 需要银行卡信息 |
| 帮你签署法律合同 | 需要法人/个人数字证书 |

### 手机/硬件
| 做不到 | 原因 |
|--------|------|
| 操作你的手机 App | 没有手机访问权限 |
| 看你的手机屏幕 | 没有屏幕共享能力 |
| 访问摄像头/麦克风 | 没有硬件访问权限 |
| 读取传感器数据（GPS/IMU等） | 没有设备传感器 |
| 蓝牙/NFC 通信 | 没有硬件 |

### 原生开发
| 做不到 | 原因 |
|--------|------|
| 编译 iOS App | 需要 Xcode + Mac + 开发者证书 |
| 编译 Android App（原生） | 需要 Android Studio + SDK |
| 写 AR/VR 应用 | 需要真机调试 ARKit/ARCore |
| 写硬件驱动 | 需要硬件 + 调试器 |

### 实时/流媒体
| 做不到 | 原因 |
|--------|------|
| 处理实时音视频流 | 无 GPU/音频硬件 |
| 语音识别（ASR） | 我只能调 API，不能自己跑模型 |
| 实时视频分析 | 无摄像头 + GPU |

### 人工操作
| 做不到 | 原因 |
|--------|------|
| 点网页上的按钮 | 没有浏览器自动化（除非你配 Playwright） |

| 接听电话 | 没有电话能力 |
| 线下跑腿 | 没有物理身体 |
