# 阿里云 ECS 部署指南

## 第一步：购买 ECS 云服务器（约5分钟）

1. 打开 [阿里云 ECS 购买页](https://ecs.console.aliyun.com/)
2. 选择以下配置（最低配即可，约 ¥50/月）：

| 配置项 | 推荐选择 |
|---|---|
| 付费方式 | 按量付费（测试用）/ 包年包月（长期用） |
| 地域 | 离你最近的（如 北京/上海） |
| 实例规格 | 2 vCPU + 2 GB 内存（最低配） |
| 操作系统 | Ubuntu 22.04 LTS |
| 系统盘 | 40 GB ESSD |
| 公网带宽 | 1 Mbps 按使用流量 |

3. **关键步骤 — 配置安全组**（让外部能访问你的网页）：

   购买时在"安全组"设置中添加一条规则：

   | 方向 | 协议 | 端口 | 授权对象 |
   |---|---|---|---|
   | 入方向 | TCP | 5000 | 0.0.0.0/0 |

4. 设置 root 密码（牢记），确认购买

---

## 第二步：SSH 登录服务器

购买成功后，在 ECS 控制台找到**公网 IP**，然后在本机终端执行：

```bash
ssh root@<你的公网IP>
```

首次登入会提示确认指纹，输入 `yes` 回车，再输入你设置的密码。

---

## 第三步：一键部署

登录服务器后，依次执行：

```bash
# 1. 先安装 git
apt-get update && apt-get install -y git

# 2. 克隆你的项目（把下面 URL 替换成你的 GitHub 仓库地址）
git clone https://github.com/chenyinquan105/-ai-openclaw-.git /opt/meituan-butler
cd /opt/meituan-butler

# 3. 配置 API Key
cp .env.example .env
vim .env    # 填入真实的 DEEPSEEK_API_KEY 和 AMAP_API_KEY

# 4. 运行部署脚本
chmod +x deploy.sh
./deploy.sh
```

---

## 第四步：打开网页

部署成功后，浏览器访问：
```
http://<你的公网IP>:5000
```

就能看到沙盒控制台了，所有功能与本地完全一致！

---

## 附加：设置为正式端口 80（可选）

如果不想在 URL 里带 `:5000`，可以配置 nginx 转发：

```bash
cat > /etc/nginx/sites-available/meituan << 'EOF'
server {
    listen 80;
    server_name _;
    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_read_timeout 120s;
    }
}
EOF

ln -sf /etc/nginx/sites-available/meituan /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t && systemctl restart nginx
```

再在安全组规则里**额外添加端口 80**，之后就能直接访问 `http://<你的IP>` 了。
