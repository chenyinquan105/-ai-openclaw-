# Product Overview

Meituan_Spatial_Butler — 基于 OpenClaw 的全天候时空沙盒数字管家。为本地生活场景提供从商户搜索、多节点排程、路径规划到实时提醒的一站式智能决策，支持异常环境下的自动容灾变轨。

## Core Capabilities

1. **多维约束商户检索** — 根据品类、评分、距离、价格、忌口等多维约束搜索商户，自动注入用户长期偏好（偏好菜系置顶、忌口隐藏、预算超标降权）
2. **并发排程引擎** — 多品类任务并发时间线求解，空间重心策略优化路径顺序，冲突检测 + CONFIRM_REQUIRED 交互闭环
3. **24H 时空沙盒仿真** — 虚拟时钟驱动的计划实时推进，支持快进/跳转/倍速，排程节点到达自动触发事件
4. **异步实时推送** — SSE + 守护线程独立轮询，五步强闭环催促（提醒→确认→延时→再催→紧急联络人），真实时间与虚拟时钟分离
5. **异常容灾变轨** — 暴雨/停电/堵车/排号异常注入后，Plan B 自动生成（SWAP/BYPASS/POSTPONE），弹窗请求用户确认

## Target Use Cases

- 用户输入「帮我安排下午理发+喝咖啡+宠物洗澡」→ 自动搜索商户 → 排程 → 路径规划 → 提醒
- 沙盒模式下拖拽时间轴快进 → 观察排程节点自动到达 → 异常注入验证容灾逻辑
- 长期运行中喝水/吃药提醒按时推送 → 45分钟无响应升级紧急联络人

## Value Proposition

- **偏好持久化**：四维度偏好谱（口味/通勤/预算/健康）自动读写，LLM 语义检测偏好变化
- **离线可复现**：所有数据（商户/天气/排队）均为本地 Mock，不依赖外部 API
- **双轨并行**：静态排程 + 沙盒仿真共存，同一套 Skill 矩阵支撑两种运行模式
- **薄桥接层**：OpenClaw Gateway ↔ Python 后端通过 HTTP Bridge 契约通信，不动现有业务代码
